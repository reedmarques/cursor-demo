# DAM Backend

RESTful API server for the Digital Asset Management system.

## Running the Server

```bash
# Install dependencies
npm install

# Start server (production)
npm start

# Start server with auto-reload (development)
npm run dev
```

Server runs on `http://localhost:5000`

## API Documentation

See main README.md for complete API endpoint documentation.

## Data Storage

Data is persisted in `data.json` file. The file is automatically created with sample data on first run.
